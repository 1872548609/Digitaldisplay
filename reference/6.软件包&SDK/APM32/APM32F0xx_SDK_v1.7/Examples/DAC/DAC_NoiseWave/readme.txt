&par Example Description

This example shows how to configure the DAC peripheral to generate NoiseWave
The waveform can be displayed using an oscilloscope. using DAC_CHANNEL_1(PA4)
to output NoiseWave.

&par Directory contents

  - DAC/DAC_NoiseWave/src/apm32f0xx_int.c     Interrupt handlers
  - DAC/DAC_NoiseWave/src/main.c              Main program

&par Hardware and Software environment

  - This example runs on APM32F072 MINI Devices.
